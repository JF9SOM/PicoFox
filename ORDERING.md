| 144MHz Band Pass Filter | 5 | $12〜$18 | $60〜$90 |

| **合計（5台分・送料込み）** | | | **$75〜$115（約11,000〜17,000円）** |

**まとめ買い推奨の理由**:
- 国際送料はほぼ固定のため、1個ずつ発注すると1個あたりの送料負担が大きい
- 5個発注で送料を5台に分散でき、1台あたりの追加コストを最小化できる
- BPFの予備（故障・紛失対策）として1〜2個多めに発注しておくと安心

**発注手順**:
1. rfpowertools.com にアクセスし "144MHz" または "2 meter BPF" で検索
2. 5個カートに追加
3. チェックアウト時に配送先を日本に設定、送料を確認
4. 支払いはクレジットカード（Visa/Mastercard）またはPayPal

> **備考**: BOM.mdでは hobbypcb.com も選択肢として記載している。rfpowertools.com で見つからない場合は hobbypcb.com の "144MHz BPF" も確認すること。

## 3. 発注の順序と優先度

リードタイムの長い海外便から先に発注する。

```
【第1週】海外発注（同時並行）
  ├─ rfpowertools.com: 144MHz BPF × 5（リードタイム 2〜3週間）
  └─ AliExpress:      DRA818V × 5    （リードタイム 2〜4週間）
                      tinySA Ultra × 1（リードタイム 2〜4週間）

【第1〜2週】国内発注（届き次第作業開始可能）
  └─ 秋月電子: Pico × 5、パッシブ部品、LiPo、ケーブル類
               → 翌日〜3日で到着

【第2週〜】到着次第作業開始
  ├─ 秋月電子部品到着 → ブレッドボードで配線確認・ファームウェア動作確認
  ├─ DRA818V到着     → TESTING.md §4 の初期確認手順を実施
  └─ BPF到着        → DRA818V + BPF でスプリアス測定（TESTING.md §3）

【第3〜4週】5局同時テスト
  └─ 全部品揃い次第 → TESTING.md §2 の5局同時起動テスト
```

**優先度まとめ**:

| 優先度 | 品目 | 理由 |
|:------:|------|------|
| ★★★ | DRA818V × 5 | 最重要部品、リードタイム長い |
| ★★★ | 144MHz BPF × 5 | 電波法上必須、リードタイム長い |
| ★★☆ | tinySA Ultra | スプリアス確認に必要 |
| ★★☆ | Pico × 5 | 開発・デバッグに即使える |
| ★☆☆ | パッシブ部品・ケーブル | 国内調達で即入手可能 |
| ★☆☆ | LiPoバッテリー | 国内調達、ただし航空便不可に注意 |

## 4. 注意事項

### LiPoバッテリーは航空便不可

- リチウムポリマーバッテリーは**国際航空便（SAL便含む）での輸送が禁止または制限**されている
- AliExpressで注文すると「船便のみ」となり**納期が6〜8週間**になる場合がある
- **国内（秋月電子・Amazonなど）での調達を強く推奨**

### tinySA Ultraのクローン品に注意

- AliExpressには公式品と外観が酷似したクローンが多数出回っている
- クローン品は測定精度・ファームウェアサポートが保証されない
- **必ず tinySA.org で公式販売店を確認してから発注する**
- 公式品には「tinySA.org」のロゴとシリアル番号が付属

### BPFは5台分まとめて発注

- rfpowertools.comへの国際送料は個数によらずほぼ固定
- 1台ずつ発注すると送料が5倍かかる
- 予備1〜2個を含め**6〜7個まとめて発注**するとコスト効率が良い

### 電波法：免許について

本機は自作のアマチュア無線機器です。日本国内で送信するには、アマチュア局免許状が必要ですが、自作機器の場合、JARDの保証認定を受けることでアマチュア局の開設または変更申請が可能です。
