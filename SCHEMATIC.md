## 3. PWM音声出力 LPF 回路図

RP2040のGPIO15から出力される10bit PWM（約122kHz）をアナログ音声に変換し、DRA818VのMICピンへ供給する。

```
                    R1            DRA818V
  GPIO15 ──────────[1kΩ]──────┬──→ MIC pin
  (PWM ~122kHz,               │
   10bit, 3.3Vp-p)          [C1]
                             [33nF]
                               │
                              GND (DRA818V GND と同一点)


  周波数特性:
  ────────────────────────────────────────────────
   周波数      ゲイン    用途
  ────────────────────────────────────────────────
   1 kHz       0 dB     モールストーン（通過）
   4.8 kHz    -3 dB     カットオフ周波数 fc
   10 kHz    -13 dB     減衰（PWM成分を抑制）
   122 kHz   -48 dB     PWMキャリア（十分減衰）
  ────────────────────────────────────────────────

  fc = 1 / (2π × 1kΩ × 33nF) ≈ 4.8 kHz

  代替値: R=1kΩ, C=47nF → fc ≈ 3.4 kHz（偏移を絞りたい場合）

  注意: C1 は DRA818V MICピンのすぐ近く（1cm以内）に配置すること。
```

## 4. SYNC・スタートボックス回路図

### 4-a. フォックス側 SYNC 入力回路（1台分）

```
  3.5mm ステレオジャック
  ┌─────────────┐
  │ Tip ────────┼──[R: 100Ω]───── GPIO13 (SYNC_PIN)
  │             │                    │
  │             │               [内部プルアップ 50kΩ]
  │             │                    │
  │ Ring ───────┼──────────────── +3V3
  │             │
  │ Sleeve ─────┼──────────────── GND
  └─────────────┘

  平常時: GPIO13 = HIGH（内部プルアップ）
  同期時: スタートボックスがTipをGNDに落とす → GPIO13 = LOW → エポックリセット
```

### 4-b. スタートボックス回路

```
                         3.5mm ジャック OUT
                         ┌─────────────┐
  +3V3 ──[10kΩ]──┬───── │ Tip         ├── ケーブルへ
                 │       │             │
                [SW1] ── │             │
  (タクトスイッチ)│       │ Ring ───────┼── GND
                GND      │ Sleeve ─────┼── GND
                         └─────────────┘

  SW1 OFF (平常): Tip = HIGH（プルアップ）
  SW1 ON  (押下): Tip = GND  → 全Foxの GPIO13 が LOW → 同期

  注: フォックス側の内部プルアップ（50kΩ）が5台並列になるため
      合成抵抗 ≈ 10kΩ。スタートボックスの外部プルアップは省略可。
```

### 4-c. 5台同時接続図（デイジーチェーン）

```
  スタートボックス
  ┌─────────────────┐
  │                 │
  │  SW1 ─── GND   │
  │   │             │
  │  [100Ω]        │
  │   │             │
  │  Tip────────────┼─── ケーブル1 ──┬── Fox1: Tip→[100Ω]→GPIO13
  │  Ring───── GND ─┼─── ケーブル1 ──┘   Fox1: Ring→GND
  │                 │
  └────────┬────────┘
           │ OUT ジャック
           │
           ├── ケーブル2 ──┬── Fox2: Tip→[100Ω]→GPIO13
           │              └── Fox2: Ring→GND
           │
           ├── ケーブル3 ──┬── Fox3: Tip→[100Ω]→GPIO13
           │              └── Fox3: Ring→GND
           │
           ├── ケーブル4 ──┬── Fox4: Tip→[100Ω]→GPIO13
           │              └── Fox4: Ring→GND
           │
           └── ケーブル5 ──┬── Fox5: Tip→[100Ω]→GPIO13
                          └── Fox5: Ring→GND

  TipラインはスタートボックスとFox5本が並列接続。
  SW1 を押すと全 Fox の GPIO13 が同時に LOW になる。
```

## 5. 電源回路図

```
  USB-C コネクタ
       │
  USBLC6-2SC6 (ESD保護)
       │
   VBUS (+5V) ──────────────────────────── RP2040 (USB電源時)
       │
  BQ21040DBVR (LiPoチャージャ)
       │ 充電電流
       ▼
  LiPo 3.7V ◄──── JST PH コネクタ ◄──── LiPo バッテリー (2000〜3000mAh)
       │
  AO3407A / AO3401A (電源スイッチ MOSFET)
       │
   SW (スライドスイッチ)
       │
  +BATT (3.7〜4.2V) ─────────────────────────────────────────────┐
       │                                                          │
  ME6217C33M5G (3.3V LDO, 最大500mA)                             │
       │                                                          │
   +3V3_INT ──→ RP2040, Si5351, W25Q128, USBLC6                  │
   (内部 3.3V)   各種プルアップ、ロジック回路                     │
                                                                  │
                                                         ┌────────▼────────┐
                                                         │  外部 3.3V 電源  │
                                                         │  (1A以上、別途)  │
                                                         │ ※DRA818V専用    │
                                                         └────────┬────────┘
                                                                  │
                                                            +3V3_DRA (3.3V / 1A)
                                                                  │
                                                             DRA818V VCC
                                                         (送信時 最大 700mA)


  デカップリング:
    +3V3_INT  各ICのVCC直近に 100nF + 1uF を配置
    +3V3_DRA  DRA818V VCCに 100uF(電解) + 100nF(セラミック) を並列配置
    +BATT     バッテリ端子近傍に 100uF を配置

  グランド接続:
    RP2040 GND ─┐
    DRA818V GND─┼── 1点でのみ接続（グランドループを防ぐ）
    BPF GND  ───┘
```

## 6. RF信号経路・アンテナ接続

```
  DRA818V
  ┌─────────────────────────┐
  │                         │
  │  内蔵 VCO + PLL          │
  │  134〜174 MHz            │
  │  内蔵 PA (≈1W / 30dBm)  │
  │                         │
  │  ANT ───────────────────┼── SMAケーブル (10cm, 50Ω) ──→ BPF IN
  │                         │
  └─────────────────────────┘

  144MHz BPF (HobbyPCB または LC自作)
  ┌──────────────────────────────┐
  │                              │
  IN ──┤ 通過帯域: 144〜146MHz    ├── OUT
  │    │ 挿入損失: ≈ 1〜2dB      │
  │    │ スプリアス抑制: > 40dBc  │
  │                              │
  └──────────────────────────────┘
       │
       └── SMAケーブル (10cm, 50Ω) ──→ SMAコネクタ (エッジマウント)
                                                │
                                          1/4λ アンテナ
                                          145MHz: 51.5cm
                                          (またはホイップ、Jポール等)

  インピーダンス: 全経路 50Ω
  BPFとSMAコネクタ間の配線は 5cm 以内を推奨。
  同軸ケーブルのシールドはグランドプレーンに確実に接続すること。


  アンテナ長の目安 (145.660 MHz):
  ────────────────────────────────
  1/4λ      = 300,000 / 145.660 / 4 ≈ 51.5 cm
  1/2λ DP   = 各エレメント ≈ 51.5 cm × 2
  5/8λ GP   ≈ 64.4 cm (利得 +3dBi)
  ────────────────────────────────
```

## 7. 完成系の信号フロー（まとめ）

```
  [LiPo] → [電源SW] → [LDO 3.3V] → [RP2040]
                                        │
                    ┌───────────────────┤
                    │                   │
              [UART 9600bps]      [PWM GPIO15]
                    │                   │
                    ▼                 [LPF]
              [DRA818V]                 │
              周波数設定 ←──────────── MIC
              PTT制御   ←── GPIO10
              PDN制御   ←── GPIO11
                    │
                   RF出力 (145.660 MHz, ≈1W)
                    │
                [144MHz BPF]
                    │
               [SMAコネクタ]
                    │
                [アンテナ]
                    ↑
              電波輻射 145.660/145.780 MHz FM


  [SYNC入力 GPIO13] ←── [スタートボックス SW1] ←── 運営者
       │
  エポックリセット → 60秒サイクル再開始 → 全Fox同期
```
